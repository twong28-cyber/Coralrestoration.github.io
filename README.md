# Coralrestoration.github.io

Welcome to the Coral Restoration project! This website and repository focus on efforts to restore coral reefs through scientific research and community involvement.

## About

This project aims to:
- Raise awareness about coral bleaching and its effects.
- Share data-driven insights on reef restoration techniques.
- Engage volunteers and researchers in coral care activities.
